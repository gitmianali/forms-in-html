What is a Form in HTML?

An HTML form is used to collect user input. It can include fields like text boxes, radio buttons, checkboxes, drop-downs, and submit buttons.

EXAMPLE

common input types

text for short text
password hides input
email for email validation
number only allows numbers
date calendar input
file upload files
checkbox select one or more options
radio select only one option
submit submit button
reset reset all form fields
